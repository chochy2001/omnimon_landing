# OmniMon Landing Page

Esta es la landing page oficial de OmniMon, construida con Astro 5 y Tailwind CSS.

## Desarrollo

1. Asegúrate de tener `bun` instalado.
2. Clona el repositorio y ubícate en este directorio.
3. Instala dependencias:
   ```bash
   bun install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   bun run dev
   ```

## Compilación

Para generar los archivos estáticos listos para producción:

```bash
bun run build
```

El resultado estará en el directorio `dist/`.

## Características
- Generación estática para máximo rendimiento y SEO.
- Dark mode por defecto.
- Estilizado moderno con Tailwind CSS (v4).
- Totalmente responsive.
